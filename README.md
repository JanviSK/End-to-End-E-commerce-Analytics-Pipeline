# End-to-End E-commerce Analytics Pipeline
Overview:
Built an end-to-end data pipeline using multi-table e-commerce data (customers, orders, products, payments) to generate actionable business insights on revenue, customer behavior, and operational efficiency.

Pipeline:
CSV → Python (ETL) → PostgreSQL (Data Warehouse & PK–FK Modeling) → dbt (Transformations) → Power BI Dashboard
Data Modeling:
Designed relational schema using primary–foreign keys
Implemented Star Schema (fact & dimension tables)
Demonstrated Snowflake schema by normalizing dimensions

Analysis:
Revenue trends & MoM growth
Customer analysis (RFM, retention, repeat rate)
Order lifecycle & approval delays
Payment behavior & installment trends
Product & category performance
Advanced: Cohort analysis, funnel analysis, unit economics

Dashboard:
Interactive Power BI dashboard with KPIs like Revenue, Orders, AOV, Growth, and Customer Insights, along with dynamic filters and drill-downs.

Tech Stack:
SQL (Joins, CTEs, Window Functions)
PostgreSQL (Data Modeling, PK–FK Relationships)
Python (Pandas) (ETL & Data Cleaning)
dbt (Data Transformation & Modeling)
Power BI (Dashboarding, DAX)
Git/GitHub, VS Code

Outcome:
Delivered a scalable analytics solution combining data engineering, modeling, and business intelligence to support data-driven decision-making.
